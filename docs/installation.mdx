## Quick Install

<Tabs items={["macOS / Linux", "Windows"]}>
  <Tab value="macOS / Linux">
    ```bash
    curl -fsSL https://athas.dev/install.sh | sh
    ```

    If you prefer `wget`:

    ```bash
    wget -qO- https://athas.dev/install.sh | sh
    ```

  </Tab>
  <Tab value="Windows">
    ```powershell
    powershell -ExecutionPolicy ByPass -c "irm https://athas.dev/install.ps1 | iex"
    ```
  </Tab>
</Tabs>

The install script detects your OS and architecture, downloads the correct
release from GitHub, verifies the SHA256 checksum, and installs it.

<Callout type="info">
You can inspect the script before running it:

```bash
curl -fsSL https://athas.dev/install.sh | less
```

</Callout>

## Manual Download

Download the latest version from GitHub releases:

**[Download from GitHub Releases](https://github.com/athasdev/athas/releases)**

<Tabs items={["macOS", "Windows", "Linux"]}>
  <Tab value="macOS">
    1. Download the `.dmg` file from [releases](https://github.com/athasdev/athas/releases)
    2. Open the downloaded file
    3. Drag Athas to your Applications folder
    4. Open Athas from Applications
  </Tab>
  <Tab value="Windows">
    1. Download the installer from [releases](https://github.com/athasdev/athas/releases)
    2. Run the installer
    3. Follow the installation wizard
    4. Launch Athas from the Start Menu
  </Tab>
  <Tab value="Linux">
    Download the appropriate package for your distribution:

    **Debian/Ubuntu (.deb):**
    ```bash
    sudo dpkg -i athas_*.deb
    ```

    **Fedora/RHEL (.rpm):**
    ```bash
    sudo rpm -i athas-*.rpm
    ```

    **AppImage:**
    ```bash
    chmod +x Athas-*.AppImage
    ./Athas-*.AppImage
    ```

  </Tab>
</Tabs>

## Uninstall

<Tabs items={["macOS", "Windows", "Linux"]}>
  <Tab value="macOS">
    ```bash
    rm -rf /Applications/Athas.app
    rm -rf ~/Library/Application\ Support/com.code.athas
    rm -rf ~/Library/Caches/com.code.athas
    ```
  </Tab>
  <Tab value="Windows">
    Uninstall from **Settings > Apps > Installed apps** or run the uninstaller
    from the Start Menu.

    To remove remaining data:
    ```powershell
    Remove-Item -Recurse -Force "$env:APPDATA\com.code.athas"
    ```

  </Tab>
  <Tab value="Linux">
    **If installed via dpkg:**
    ```bash
    sudo dpkg -r athas
    ```

    **If installed via the install script:**
    ```bash
    rm ~/.local/bin/athas
    ```

    Remove application data:
    ```bash
    rm -rf ~/.local/share/com.code.athas
    rm -rf ~/.config/com.code.athas
    ```

  </Tab>
</Tabs>
